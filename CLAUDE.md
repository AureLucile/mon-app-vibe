# CLAUDE.md

## Règles du projet

- Après chaque modification de code, créer systématiquement un commit avec un message descriptif (Conventional Commits : feat:, fix:, refactor:, docs:, etc.).

## Commandes slash

- `/ship` — Stage toutes les modifications, crée un commit avec un message Conventional Commits descriptif, et push sur origin main. Équivalent de `git add -A && git commit && git push origin main` en une seule commande.
